# Automated-Serverless-Contact-Management-System
A Serverless Contact Management &amp; Ebook Delivery System featuring a secure, HTTPS-only frontend (S3/CloudFront) and a resilient backend (API Gateway, Lambda, DynamoDB, SES). The entire infrastructure is managed via Infrastructure as Code (CloudFormation) and deployed through a full AWS CI/CD pipeline (CodePipeline/CodeBuild).
